Cell Differentiation Study

Poisson GLM with Interaction Effects

Published

January 21, 2026

Overview

This script demonstrates:

  • Poisson regression for count data (cell differentiation)
  • Testing synergistic vs. additive effects of two treatment factors
  • Model diagnostics using deviance and Pearson statistics
  • Dispersion estimation for overdispersion assessment

The study investigates whether TNF (Tumor Necrosis Factor) and IFN (Interferon) agents stimulate cell differentiation synergistically or independently.

Poisson GLM: Main Effects + Interaction

We fit a Poisson GLM with the interaction term to test for synergistic effects:

# Fit Poisson GLM
glm_poisson <- glm(
  formula = y ~ TNF * IFN,
  family = poisson,
  data = cells
)

Model Interpretation

The terms in the model are: - Intercept: Baseline log count (reference group: no TNF, no IFN) - TNF: Main effect of TNF treatment - IFN: Main effect of IFN treatment - TNF:IFN: Interaction effect (synergy test)

Goodness-of-Fit Testing

Two key statistics for assessing model fit in Poisson regression:

1. Deviance Test:

p_value_deviance <- 1 - pchisq(glm_poisson$deviance, df = glm_poisson$df.residual)
cat("Deviance:", round(glm_poisson$deviance, 3), "\n")
Deviance: 142.385 
cat("p-value:", round(p_value_deviance, 4), "\n")
p-value: 0

2. Pearson Chi-Square Test:

# Calculate Pearson residuals
pearson_residuals <- resid(glm_poisson, type = "pearson")
pearson_stat <- sum(pearson_residuals^2)

p_value_pearson <- 1 - pchisq(pearson_stat, df = glm_poisson$df.residual)
cat("Pearson Chi-Square:", round(pearson_stat, 3), "\n")
Pearson Chi-Square: 140.822 
cat("p-value:", round(p_value_pearson, 4), "\n")
p-value: 0

Overdispersion Assessment

Poisson regression assumes the mean equals the variance. When this fails, we have overdispersion.

# Estimated dispersion parameter
dispersion_pearson <- pearson_stat / glm_poisson$df.residual
dispersion_deviance <- glm_poisson$deviance / glm_poisson$df.residual

cat("Dispersion (Pearson):", round(dispersion_pearson, 4), "\n")
Dispersion (Pearson): 11.7352 
cat("Dispersion (Deviance):", round(dispersion_deviance, 4), "\n")
Dispersion (Deviance): 11.8654 
if (dispersion_pearson > 1) {
  cat("\nOverdispersion detected!\n")
  cat("Consider using quasi-Poisson or negative binomial model.\n")
}

Key Findings

  1. Model Assessment: Evaluate whether the Poisson assumption is appropriate via goodness-of-fit tests
  2. Interaction: Test the TNF:IFN interaction to determine if effects are synergistic
  3. Overdispersion: If present, consider alternative models (quasi-Poisson, negative binomial)
  4. Interpretation: Rate ratios provide intuitive interpretation of treatment effects

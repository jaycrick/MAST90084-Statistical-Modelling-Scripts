Verification: MLE Formula

For exponential distribution, MLE of lambda is # events / total time at risk:

Code 
sum(bcancer$status[bcancer$group == "N"]) /
    sum(bcancer$time[bcancer$group == "N"])
[1] 0.003026634
Code 
sum(bcancer$status[bcancer$group == "P"]) /
    sum(bcancer$time[bcancer$group == "P"])
[1] 0.007838746

Delta Method Confidence Interval

Code 
beta <- cancer_exp$coeff[2]
var_beta <- (cancer_exp$var)[2, 2]
var_delta <- (exp(-beta))^2 * var_beta

exp(-beta) + qnorm(0.975) * sqrt(var_delta)
  groupP 
5.115883
Code 
exp(-beta) - qnorm(0.975) * sqrt(var_delta)
    groupP 
0.06396006

The lower bound doesn’t cross to the negative number region, which is lucky.

Alternative using monotone transformation:

Code 
exp(-(beta - qnorm(0.975) * sqrt(var_beta)))
  groupP 
6.868405
Code 
exp(-(beta + qnorm(0.975) * sqrt(var_beta)))
   groupP 
0.9766015

This guarantees that the lower bound doesn’t cross to the negative number region.

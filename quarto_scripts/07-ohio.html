Ohio Respiratory Data: GEE Analysis

Generalized Estimating Equations with Repeated Measures

Published

March 12, 2026

Overview

This script aims to reproduce the results in Example 6.4 of Fahrmeir & Tutz using the Ohio respiratory dataset.

Packages and Setup

library(conflicted)
library(gee)
library(geepack)

Scale Parameter

Code 
summary(geefit_indep)$scale
[1] 1.003738

Now we tease out the Pearson residuals and compute the method of moment dispersion estimator by hand:

Code 
pearson_resid <- (ohio$resp - geefit_indep$fitted) /
    sqrt(
        geefit_indep$fitted.values * (1 - geefit_indep$fitted.values)
    )

scale_by_hand <- sum(pearson_resid^2) /
    (geefit_indep$nobs - length(geefit_indep$coeff))

scale_by_hand
[1] 1.003738

Same number as the one reported in summary.

Beta Estimate for Age = 10

With effect coding, age=10 is minus the sum of the other three age coefficients:

Code 
beta_age_10 <- -sum(summary(geefit_indep)$coeff[
    c("age1", "age2", "age3"),
    1
])

[1] -0.2827363

Same as the book.

Code 
pearson_resid <- (ohio$resp - geefit_indep$fitted) /
    sqrt(
        geefit_indep$fitted.values * (1 - geefit_indep$fitted.values)
    )

scale_by_hand <- sum(pearson_resid^2) /
    (geefit_indep$nobs - length(geefit_indep$coeff))

Linear Mixed Effects Models

Random Intercept and Random Slope Models

Published

March 12, 2026

Overview

This script demonstrates linear random effect models using the lme4 package.

Paper for lme4 package: https://cran.r-project.org/web/packages/lme4/vignettes/lmer.pdf

Code 
mmod <- lmer(bright ~ 1 + (1 | operator), pulp)

Note that lmer doesn’t report p-values for their estimates. The between subject SD 0.21 of the MLE method is smaller than the between subject SD 0.26 of the REML method because MLE tends to bias the estimates towards zero.

Code

(Intercept)

Code 
fixef(smod)
(Intercept) 
       60.4

The fixed effect estimates of the two methods happen to be the same, but they should generally differ.

Hand-Computed Fixed Effect Estimate

Code 
components <- as.data.frame(VarCorr(mmod))$vcov
Vi <- components[2] * diag(5) + components[1] * tcrossprod(rep(1, 5))

V <- Matrix::bdiag(Vi, Vi, Vi, Vi)
Z <- rep(1, 20)

Hand-computed fixed effect estimate:

Code 
(1 / (t(Z) %*% solve(V) %*% Z)) * t(Z) %*% solve(V) %*% pulp$bright
1 x 1 Matrix of class "dgeMatrix"
     [,1]
[1,] 60.4

Hand-computed standard error for fixed effect (the usual covariance matrix for the GLS (general least square) estimate):

Code 
sqrt(1 / (t(Z) %*% solve(V) %*% Z))
1 x 1 Matrix of class "dgeMatrix"
          [,1]
[1,] 0.1494434

These numbers check out with ones reported in summary(mmod)$coeff.

Model Diagnostics

One can also plot the residuals as model diagnostics. The default residuals are computed using the estimated random effects. This means these residuals can be regarded as estimates of epsilons which is usually what we want:

Code 
par(mfrow = c(1, 2))
qqnorm(residuals(mmod), main = "")
plot(fitted(mmod), residuals(mmod), xlab = "Fitted", ylab = "Residuals")
abline(h = 0)
Code 
dev.off()
null device 
          1

Compute the residual by hand:

Code 
residual_manual <- pulp$bright -
    rep(ranef(mmod)$operator[, 1], each = 5) -
    fixef(mmod)

They check out with the residual reported:

Code

psid$cyear <- psid$year - 78

As explained by Faraway, the year is centered at the year 1978 so that the intercept will predict the log income in 1978, but not 1900!

Code 
mmod <- lmer(log(income) ~ cyear * sex + age + educ + (cyear | person), psid)

smod <- lmer(
    log(income) ~ cyear * sex + age + educ + (cyear | person),
    psid,
    REML = FALSE
)

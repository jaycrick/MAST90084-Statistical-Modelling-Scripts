GLM with Mixed Effects

Binomial GLMM on Ohio Respiratory Data and Ordinal Mixed Models

Published

March 12, 2026

Overview

This script demonstrates Generalized Linear Mixed Models (GLMM) for handling subject-specific random effects in longitudinal and ordinal data.

Methods covered: - Laplace approximation via glmer() (lme4) - Adaptive Gauss-Hermite quadrature (AGHQ) for improved accuracy - Penalized Quasi-Likelihood (PQL) via glmmPQL() (MASS) - Ordinal mixed models via clmm() (ordinal)

Available GLMM Software Comparison

Many R packages exist for fitting GLMMs, using different estimation methods:

  • MASS::glmmPQL: Penalized quasi-likelihood
  • lme4::glmer: Laplace approximation and adaptive Gauss-Hermite quadrature (AGHQ)
  • MCMCglmm: Markov chain Monte Carlo
  • glmmML: AGHQ
  • glmmADMB: Laplace
  • glmm (from Jim Lindsey’s repeated package): AGHQ
  • ordinal::clmm: For ordinal outcomes with random effects
  • gamlss.mx, ASREML-R, sabreR, mcemGLM: Other specialized approaches

We’ll focus on the most commonly used: glmer(), glmmPQL(), and clmm().

GLMM via Laplace Approximation

The default glmer() approach uses the Laplace approximation, which is fast and reasonably accurate. The Nelder-Mead optimizer is recommended by Faraway (2016):

Code 
glmer_fit <- glmer(
    resp ~ age * smoke + (1 | id),
    data = ohio,
    family = binomial,
    control = glmerControl(optimizer = "Nelder_Mead")
)

Key Learning Points

  1. Subject-specific vs. population-average: GLMM coefficients are conditional (subject-specific), differing from GEE marginal effects
  2. Estimation methods:
    • Laplace: Fast, reasonable accuracy
    • AGHQ: Slower, higher accuracy (more quadrature points = better approximation)
    • PQL: Fastest, lower accuracy (avoids integration entirely)
  3. Random effects: Represent between-subject heterogeneity; extracted as conditional modes
  4. Ordinal mixed models: Use clmm() for cumulative link models with random effects
  5. Contrast coding: Effect coding vs. dummy coding changes coefficient interpretation but not model fit
